Potential Questions for Cover Songs One-day

Each question below contains has a snippet from a song that went on to have a well-known cover version by a different artist. We want the name of the artist that released the cover version, not the artist performing the recorded sample. Note that some songs have inspired multiple well-known covers, so pay attention to the year of the cover given in the question which sould narrow it down to a single cover.

For example, if the question states 'Original: 1974; Cover: 1992.' and had this clip:

You should listen to that sample and figure out that it is the song I Will Always Love You (originally released by Dolly Parton in 1974). In 1992, it was covered in by Whitney Houston. So the correct answer should be "Whitney Houston".
